# 2023.4.23

# 2023.4.25
敏感是一种天赋，你们也终将感受到敏感是你的天赋。
我以前总以为生活还有别处，漫不经心、满不在乎，可是生活就在此处，人们就在此处，你就在此处。
在面对群体性的苦难、压迫和谋杀的时候，那不是冷静，那是冷血。那不是理性，那是奴性。敏感不只是一种天赋，敏感是一种正义感。这证明你是一个不会听从一切压迫和统治的人。
