# 5.10

音乐方面：
辛爽导演最初是做乐队的，做过鼓手和吉他手。后来去光线传媒做过音乐总监。所以他有很好的音乐素养。他找了很多很小众却很有质感的音乐。他的片尾曲，每一集都是不一样的。他的每一集的片尾都是与剧情非常吻合的，就是在剧情进行到快结尾的时候，渐进式的进入音乐。让你非常沉浸，让我这种从来不看片尾的人，竟然能默默盯着屏幕，听完片尾曲。尤其是最后一集，他选了唯一一个大众的曲目，《再回首》，我一下子就被击中了，潸然泪下。
# 5.11
摄影、剪辑方面：
调色、构图、画面、光线这些也非常有质地有美感。片中他是有三条时间线，他有过几次不同时间线的转镜用了非常牛的剪辑。人作为主体，忽然转化了下角度，完全进入了另一个空间。很精妙。这个在国产电视剧里应该是没有的。

智、仁、勇是儒家所提倡的三种德行。《论语·子罕》：“子曰：知者不惑，仁者不忧，勇者不惧。”知，用同“ 智 ”。《史记·平津侯主父列传》：“智、仁、勇，此三者天下之通德。”
三达德出自《中庸》，三达德即指“智”、“仁”、“勇”此三大品行。《中庸》第二十章阐明道：“天下之达道五，所以行之者三。曰：君臣也，父子也，夫妇也，兄弟也，朋友之交也。五者，天下之达道也。知、仁、勇三者，天下之达德也，所以行之者一也。
