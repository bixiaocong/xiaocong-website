生活碎碎念

这是个没有什么章法的碎碎念，主要是为了记录下生活，顺便持续锻炼下自己的叙事能力。不限字数，不限文体，不限脏话。

你的孩子其实不是你的孩子​

你的孩子，其实不是你的孩子，他们是生命对于自身渴望而诞生的孩子。

他们通过你来到这世界，却非因你而来，他们在你身边，却并不属于你。

你可以给予他们的是你的爱，却不是你的想法，因为他们自己有自己的思想。

你可以庇护的是他们的身体，却不是他们的灵魂，因为他们的灵魂属于明天，属于你做梦也无法达到的明天。

你可以拼尽全力，变得象他们一样，却不要让他们变得和你一样，因为生命不会后退，也不在过去停留。

你是弓，儿女是从你那里射出的箭。

弓箭手望着未来之路上的箭靶，他用尽力气将你拉开，使他的箭射得又快又远。

怀着快乐的心情，在弓箭手的手里弯曲吧，因为他爱一路飞翔的箭，也爱无比稳定的弓。

5.7

最近对某部剧非常感兴趣，虽然它很不完美，但优点非常突出。想做个视频，对其进行下推广和解读。前两天发了个朋友圈力荐，最近跟朋友聊天，发现真的有人因为我的朋友圈看了这部剧。哈，星星之火，用微光来照亮你！接下来的几天写稿子吧！

5.8

大家好，今天想给大家推荐一部剧。这部剧是在五一期间，腾讯视频上热映的。目前的热度已经过去了，但是我跟朋友们交流，还是有很多人没有看过，所以我觉得有必要在推荐一下。 这部剧，目前豆瓣评分9.4分。是近几年来评分最高的国产剧了。该部剧的导演是辛爽，也是前几年的大爆剧《隐秘的角落》的导演。同时，这部剧的制作团队也是《隐秘的角落》的制作团队。所以也会看到很多的同款演员。

5.9

这部剧的背景是上个世纪、九十年代的下岗潮。以一个悬疑的凶杀案作为外壳，实际上讲的是被历史巨轮裹挟着的的个人命运。这部剧无论在剧本本身、还是音乐上、剪辑上、演员演技各个方面都做得非常棒！

// 不论年轻时多意气风发，最终都逃不过英雄迟暮，无力回天的宿命。

5.10

音乐方面： 辛爽导演最初是做乐队的，做过鼓手和吉他手。后来去光线传媒做过音乐总监。所以他有很好的音乐素养。他找了很多很小众却很有质感的音乐。他的片尾曲，每一集都是不一样的。他的每一集的片尾都是与剧情非常吻合的，就是在剧情进行到快结尾的时候，渐进式的进入音乐。让你非常沉浸，让我这种从来不看片尾的人，竟然能默默盯着屏幕，听完片尾曲。尤其是最后一集，他选了唯一一个大众的曲目，《再回首》，我一下子就被击中了，潸然泪下。

5.11

摄影、剪辑方面： 调色、构图、画面、光线这些也非常有质地有美感。片中他是有三条时间线，他有过几次不同时间线的转镜用了非常牛的剪辑。人作为主体，忽然转化了下角度，完全进入了另一个空间。很精妙。这个在国产电视剧里应该是没有的。

5.12

演员演技 除了三大主演：范伟、秦昊、陈明昊表现得非常棒，一些配角 刘琳、侯岩松（剧里最坏的那个人）演技也很好。他已经关闭了微博评论了。我还要提到两个我觉得之前不认识却演技值得一提的。 一个是演 美素的 那个演员，演的超级好，评价别人是演技封神，对她的评价就觉得她就是活在戏里的人，就完全没办法用演技来形容，因为她几乎没有表演痕迹。后来才知道，这个演员其实已经旅居海外很多年，拍一些美剧。没想到来拍一部东北题材的剧，竟然这么贴近角色。 一个是演小哑巴的蒋齐名，前一段还有他的一部电影上映<宇宙探索编辑部>。我觉得他在这部剧里的年轻演员里，是演的最好的。特别有真实聋哑人的状态。我听了一次对他的播客采访，知道他之前一直是在话剧舞台上打磨多年的演员。他在采访的对答过程中，你会发现他有非常清晰的表达，读过很多书，对角色对剧本都有自己比较深刻的见解。总之，绝对不是一个绝望的文盲。

5.16

继续稿子： 再说一下文学方面，这部剧的文学策划叫 班宇。是东北文艺复兴的代表人物之一。他就是从小生活在，沈阳的铁西区。应该是当年东北下岗潮的亲历者。最近我也看了他的《冬泳》，基本每个小故事都是以当年 下岗作为背景，体现这个事件带给这些人的伤痛。对于他们命运的改变。这部剧的内核也是这些。我觉得应该感谢这个作品，他非常正视这段历史，并且以悲悯的笔触来描写所有人的命运。而不是传统主旋律里宣扬牺牲、精神鼓励。

5.17

今天忽然发现，爷爷为三个儿子起的名字居然很有考究。 智、仁、勇是儒家所提倡的三种德行。《论语·子罕》：“子曰：知者不惑，仁者不忧，勇者不惧。”知，用同“ 智 ”。《史记·平津侯主父列传》：“智、仁、勇，此三者天下之通德。” 三达德出自《中庸》，三达德即指“智”、“仁”、“勇”此三大品行。《中庸》第二十章阐明道：“天下之达道五，所以行之者三。曰：君臣也，父子也，夫妇也，兄弟也，朋友之交也。五者，天下之达道也。知、仁、勇三者，天下之达德也，所以行之者一也。
