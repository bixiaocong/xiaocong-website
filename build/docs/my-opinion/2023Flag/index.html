带父母旅游​

之前有过一次带全家旅行的经历，发现父母对这件事不再抗拒，甚至有点期待。我觉得全家旅行这件事对父母来说意义非凡，一家人整整齐齐旅游是一种情调，是亲情的另外一种维系方式。一家人大部分时间都过得千篇一律。一起去探索新的风景，真的是脱离平淡生活的绝佳体验。
