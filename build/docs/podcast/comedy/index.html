喜剧类播客推荐

一、谐星聊天会​

简介​

[谐星聊天会]是由单立人喜剧出品的线上播客节目,每场邀请数十名观众与来自单立人的喜剧演员们一起,聊聊主题相关的故事与经历,用幽默开解生活中的沉杂琐事。

推荐理由​

与普通的播客不同，谐星聊天会的不断迭代开创了一系列的播客产品形式。现场观众和主持人间的互动，既有单口喜剧的互动特色，又塑造了出了一批素人声音，也拓宽了节目的边界。目前常驻的喜剧演员有主持人毛冬，他参加过奇葩说和脱口秀大会。还有某届的脱口秀大王，周奇墨。还有一年一度喜剧大赛的编剧六兽等等。有这些资深喜剧人士坐镇，这个播客真的不容错过。

收听渠道​

  • 小宇宙
  • 喜马拉雅

二、无聊斋​

简介​

[无聊斋]是由单口喜剧演员刘旸教主、六兽共同主持的脱口秀播客节目，每期会邀请1-2位嘉宾，在这里聊奇人异士、旅行出游、人生感悟、时事热点、电影热剧等等话题。有时候，无聊，才是最好的生活方式～

推荐理由​

同样由单立人出品，同样有六兽的参与。当然教主也很厉害，跟教主聊天永远不会尬，而且很自然，很幽默，潜移默化。社会热点在这里轻松自然的展开，人文地理在这里生动幽默的解说，爽朗的笑声使整个局面不拘束，人生百态都在“无聊斋”嘻说笑谈中。

收听渠道​

  • 小宇宙
  • 喜马拉雅
  • 蜻蜓FM

三、一言不合​

简介​

“一言不合”是单立人喜剧创办的脱口秀类网络播客节目，每期有三到四位单立人喜剧演员畅聊各类话题，说出我们自己的观点，不盲从不愤怒，亦庄亦谐，引发思考。就像石老板说的那样“让我把我的眼睛借给你，让你看看我眼中的世界。”

推荐理由​

也是由单立人喜剧厂牌创立的，主要围绕着单口喜剧演员的日常，由厂牌内部的演员来一起讨论时事、电影、书籍等话题。

收听渠道​

  • 小宇宙

四、从段子到段子​

简介​

一档没有《脱口秀大会》幽默的访谈播客。每期会请一位《脱口秀大会》上讲段子的演员，聊聊他们的创作和生活。

推荐理由​

最近《脱口秀大会》又开播了，虽然网上争议较多，但仍然不乏一些精彩演员的精彩片段。目前播客请过的嘉宾有黄大妈、House、毛豆、颜怡颜悦、江梓浩、鸟鸟和孟川。 目前还在更新中，对这些演员感兴趣的朋友可以去听听。

收听渠道​

  • 小宇宙

喜剧类的播客还有很多，比如笑果文化出的《效果编剧活动中心》、《笑果小酒馆》、《车间访谈》、《笑果FM》，还有米未传媒出的《东七门》等。大家有兴趣都可以找一下资源。
